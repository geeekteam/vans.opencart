# AUTOMATIC INSTALLATION (OCMOD)
-------------------------------------------------------------------------
# ATTENTION: you need OC2 Extension Installer configured properly and working
-------------------------------------------------------------------------
## Installation
-------------------------------------------------------------------------
* Go to Admin backend:
* install extension newsblog.xxxxxxxx.opencart.xxxx.ocmod.zip,
* go to the "Extensions / Modifications" menu and click "Refresh" button,
* add new categories (/admin/index.php?route=newsblog/category),
* add new articles (/admin/index.php?route=newsblog/article),
* add new module (/admin/index.php?route=extension/module/install&extension=newsblog_articles),

Category on your site: yourstore.com/index.php?route=newsblog/category&newsblog_category_id=ID
or with seo_url: yourstore.com/category_seo_url

Article on your site: yourstore.com/index.php?route=newsblog/article&newsblog_article_id=ID
or with seo_url: yourstore.com/category_seo_url/article_seo_url
